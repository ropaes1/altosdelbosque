## About

Altos del Bosque is a private 6-hectare residential development in Río Amarillo, Cuenca, Ecuador — just 10 minutes from the UNESCO World Heritage historic center. The development offers premium lots from 700 m² with full infrastructure, panoramic Andes views, and approved building permits.

## The Website

A single-page luxury real estate website built with:
- Pure HTML, CSS, and vanilla JavaScript (no dependencies)
- Responsive design (mobile, tablet, desktop)
- Scroll-triggered animations
- Contact form with mailto fallback
- Optimized for GitHub Pages deployment

## Deployment

This site is configured for **GitHub Pages**. To deploy:

1. Go to **Settings → Pages** in your GitHub repository
2. Under **Source**, select `main` branch and `/ (root)` folder
3. Click **Save**
4. Your site will be live at `https://[username].github.io/altosdelbosque/`
